Tentang SIMAS KOESMA

Sistem Informasi Manajemen Arsip Surat (SIMAS) adalah platform digital terintegrasi RSUD dr. R. Koesma untuk mengelola persuratan secara paperless, cepat, dan akuntabel.

Eksekusi

Menu ini biasanya hanya dapat diakses oleh user dengan Role Admin atau Petugas Loket Surat.
1. Input Data

Isi form selengkap mungkin. Nomor Agenda akan terisi otomatis. Pastikan Tanggal Surat tidak lebih baru dari Tanggal Terima.

2. Upload File

Format: PDF, JPG, PNG. Maksimal 5MB. Kompres file jika terlalu besar.

3. Kode Klarifikasi

Gunakan fitur Autocomplete. Ketik kode (misal "KM") dan pilih dari daftar.

4. Edit Data

Salah input? Gunakan menu Arsip Masuk, klik tombol (Edit). Hanya Admin yang berwenang.

Modul ini digunakan untuk mencatat surat yang dibuat oleh internal RSUD.

Fitur "Booking Nomor"

Surat fisik belum siap tapi butuh nomor agenda? Gunakan tombol Booking. Sistem akan "mengamankan" nomor urut untuk Anda. Lengkapi data nanti saat surat sudah ditandatangani.

Cara Disposisi
  1. Masuk ke menu Inbox.
  2. Klik tombol Proses (Kuning).
  3. Centang tujuan bawahan (Bisa Multi-select).
  4. Isi instruksi dan Klik Kirim.
Fitur Delegasi

Memungkinkan Sekretaris mengakses Inbox Pimpinan (misal: Sekretaris Direktur membuka Inbox Direktur). Menu Inbox Delegasi akan muncul di sidebar jika Anda memiliki akses ini.

Semua surat yang sudah diproses akan masuk ke menu Arsip.

  • Arsip Masuk: Daftar semua surat masuk RSUD.
  • Arsip Keluar: Daftar surat keluar internal.
  • Arsipku: Daftar surat yang spesifik pernah Anda tindak lanjuti (Selesai).

Gunakan menu Pencarian Lanjut di sidebar untuk mencari berdasarkan rentang tanggal atau kata kunci konten (OCR).

10 FAQ Utama

Sistem SIMAS KOESMA menggunakan teknologi Single Sign-On (SSO) Google. Artinya, kami tidak menyimpan password Anda secara langsung dalam database aplikasi.

Solusi:
  • Jika Anda lupa password, silakan lakukan reset password pada akun Gmail/Google Anda melalui halaman Google Account Recovery.
  • Setelah password Gmail berhasil diubah, silakan kembali ke halaman login aplikasi ini dan masuk menggunakan akun Google Anda seperti biasa.

Aplikasi ini menggunakan sistem Role-Based Access Control (RBAC). Menu tertentu hanya ditampilkan untuk pengguna yang memiliki wewenang (Permissions) yang sesuai.

Penyebab:
  • Anda terdaftar sebagai "Staf Pelaksana" biasa yang tidak memiliki hak input.
  • Admin belum mengaktifkan permission input_masuk pada akun Anda.
Solusi: Hubungi Admin Bagian Administrasi dan Umum untuk meminta penambahan hak akses jika memang diperlukan untuk tugas Anda.

Demi menjaga integritas dan jejak audit (audit trail) surat, sistem sengaja tidak mengizinkan user biasa untuk menarik kembali disposisi yang sudah terkirim.

Solusi:
  • Segera hubungi Admin Bagian Administrasi.
  • Berikan Nomor Agenda surat yang salah.
  • Admin akan menggunakan menu khusus "Edit Tujuan Disposisi" (ikon rantai merah di Arsip Masuk) untuk mengoreksi alur surat tersebut secara administratif.

Kegagalan upload biasanya disebabkan oleh validasi sistem atau masalah koneksi.

Cek Poin Berikut:
  1. Ukuran File: Maksimal 5MB. Jika file PDF hasil scan terlalu besar, gunakan tools kompresi PDF online terlebih dahulu.
  2. Nama File: Hindari karakter simbol aneh seperti / \ : * ? " < > | pada nama file. Gunakan nama sederhana, misal Surat_Dinas_Kesehatan.pdf.
  3. Koneksi: Pastikan internet stabil. Upload file membutuhkan bandwidth upload yang cukup.
  4. Storage Penuh: Dalam kasus jarang, penyimpanan Google Drive admin mungkin penuh. Hubungi IT untuk mengecek status "Storage Node A/B/C".

Sistem mencatat setiap pergerakan surat secara detail dalam log history.

Caranya:

  • Cari surat yang dimaksud.

Sistem mengirim notifikasi otomatis ke nomor yang terdaftar di profil pengguna.

Langkah Pengecekan:
  • Klik menu Profil Saya di sidebar.
  • Cek kolom "Nomor WhatsApp".
  • Pastikan format nomor benar. Gunakan format Indonesia standar (contoh: 08123456789 atau 628123456789). Jangan gunakan spasi atau strip.
  • Jika nomor sudah benar tapi belum masuk, kemungkinan server Gateway WA sedang sibuk/antri. Cek kembali beberapa saat lagi.

Anda tidak perlu membuat lembar disposisi manual. Sistem akan men-generate otomatis lengkap dengan QR Code.

Langkah:
  • Buka menu Arsip Masuk.
  • Cari surat yang ingin dicetak.
  • Klik tombol ikon Printer () di kolom Aksi.
  • Tunggu sejenak, sistem akan mengunduh file PDF "Lembar Disposisi" yang siap dicetak.

Nomor agenda adalah Primary Key (kunci utama) dalam database surat yang bersifat unik dan berurutan. Mengubahnya sembarangan akan merusak urutan database.

Solusi:
  • User biasa DILARANG dan tidak bisa mengubah nomor agenda.
  • Segera lapor ke Admin IT / Bagian Umum. Admin akan melakukan koreksi langsung melalui database backend jika memang terjadi kesalahan fatal (misal: lompat nomor).

Fitur ini berguna saat Anda butuh nomor surat sekarang, tapi surat fisiknya belum siap atau belum ditandatangani.

Langkah:
  • Buka menu Surat Keluar.
  • Klik tombol kuning "Booking" di samping kolom Agenda.
  • Isi catatan singkat (misal: "Untuk SK Direktur").
  • Sistem akan memberikan nomor urut. Nomor ini sekarang "milik Anda".
  • Saat surat fisik siap, buka Arsip Keluar, cari nomor booking tadi, lalu klik Edit untuk melengkapi data (upload file, perihal asli, dll).

Ya, SIMAS KOESMA adalah aplikasi berbasis web (Web App) yang Responsif. Tampilan akan otomatis menyesuaikan ukuran layar perangkat Anda.

Cara Akses:
  • Buka browser (Google Chrome atau Safari) di HP/Tablet Anda.
  • Akses URL aplikasi.
  • Anda bisa melakukan disposisi, mengecek inbox, atau melihat arsip layaknya di komputer.
